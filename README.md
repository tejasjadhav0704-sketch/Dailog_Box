At first I thought:
"Bro, it's just a popup."

* Simple Dialog worked

* Multi Choice Dialog selected everything except my patience
* Custom Dialog looked nothing like the Figma design

And then began the legendary battle of:

```text
marginStart
translationX
elevation
ConstraintLayout
```

Eventually everything started working 😌
